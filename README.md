
## About the App

This app is a text based app that writes  the response for every query that you ask the OpenAI or Chatgpt in general to add the content in the file and stores 
it in the specified folder.
This app is Text based CLI and currently requires the user to input the file name that you want to see in which the data gets stored.

## Motivation
Lately I have been finding it difficult to structure my content inside a file. Whenever, I need to revise on some topic or create a roadmap about a particular topic I need to study,
I always have to look into Gemini or Claude or ChatGPT to look into the content and that makes it time consuming.
That's why I came up with the idea of developing an agent that writes all the contents into one file and moreover one folder where I can easily access contents without delays.
